📋 Instrucciones de configuración

  1. Abrí tu Google Sheets y andá a Extensiones → Apps Script
  2. Pegá el código del archivo apps-script.gs que te generé
  3. Hacé clic en Implementar → Nueva implementación
  4. Tipo: Aplicación web | Acceso: Cualquier usuario
  5. Copiá la URL generada y pegala arriba
  6. En tu Sheets, creá las solapas: CLIENTES, SITIOS, CUADRILLAS, ASIGNACIONES
